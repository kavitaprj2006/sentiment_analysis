# IMDb Sentiment Analysis

## Project Overview
This project performs sentiment analysis on movie reviews using Natural Language Processing (NLP) techniques.
The model predicts whether a movie review is Positive or Negative.

## Dataset
IMDb Movie Reviews Dataset (50,000 reviews)

## Technologies Used
* Python
* Pandas
* NLTK
* Scikit-learn
* TF-IDF Vectorizer
* Logistic Regression
* Matplotlib

## Features
* Text preprocessing
* Stopword removal
* TF-IDF vectorization
* Sentiment prediction
* Confidence score prediction
* Confusion Matrix visualization

## Model Performance
Accuracy: 89.25%

## Project Workflow
1. Load IMDb Dataset
2. Clean review text
3. Remove stopwords
4. Convert text into TF-IDF vectors
5. Train Logistic Regression model
6. Predict sentiment
7. Evaluate using Accuracy and Confusion Matrix
